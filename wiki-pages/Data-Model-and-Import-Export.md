# 数据结构与导入导出

## 存储位置

- 使用浏览器 `localStorage`
- 默认键名：`moodDiary_v5`

## 单条记录结构

```json
{
  "id": 1741340000000,
  "mood": "happy",
  "text": "今天完成了一个小目标",
  "dk": "2026-03-07",
  "ds": "03/07 21:30"
}
```

## 字段说明

- `id`: 记录唯一标识（数字）
- `mood`: 情绪键（`happy`/`calm`/`sad`/`anxious`/`angry`/`tired`）
- `text`: 日记文本
- `dk`: 日期键（`YYYY-MM-DD`）
- `ds`: 显示用日期文本

## 导入规则

- 导入文件根节点必须是数组
- 无效记录会被过滤
- 缺失或非法 `id` 会自动补齐并去重
- 缺失 `ds` 会自动生成显示时间
