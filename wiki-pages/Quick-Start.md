# 快速开始

## 1. 克隆仓库

```bash
git clone https://github.com/leeli53/emotion-diary.git
cd emotion-diary
```

## 2. 运行项目

方式 A：直接双击 `index.html`

方式 B：起本地服务（推荐）

```bash
python3 -m http.server 8080
```

浏览器打开：

```text
http://localhost:8080
```

## 3. 基础使用流程

1. 在首页选择一个情绪
2. 输入今天的感受并保存
3. 到“时间轴”查看历史
4. 到“统计”看趋势
5. 在“设置”导出 JSON 做备份
