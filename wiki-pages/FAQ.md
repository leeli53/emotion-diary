# FAQ

## Q1: 为什么 GitHub 语言统计主要显示 HTML/CSS？

A: 语言统计由 GitHub 自己按仓库文件计算，并且有缓存。已经拆分为 `index.html`、`styles.css`、`script.js`，刷新需要一点时间。

## Q2: 数据会上传到服务器吗？

A: 不会。默认只存浏览器本地 `localStorage`。

## Q3: 更换浏览器后记录还在吗？

A: 不一定。本地存储与浏览器环境绑定。请先导出 JSON 备份。

## Q4: 导入失败怎么办？

A: 先检查 JSON 是否是数组格式，并确认字段包含 `mood` 和 `dk`。也可以先导出一份作为模板再编辑。
